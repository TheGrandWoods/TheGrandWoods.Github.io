GW Framework Plugin v3

GW v3 ports the full UI rendering and input plumbing from t2i-framework-plugin-v2 and t2i-framework-plugin-moss. It renders Perchance-style userInputs lists into a usable control panel, supports variable parsing, remembers inputs, and wires generation and sharing flows.

Key Features

  • Full `userInputs` rendering (paragraphs, selects, checkboxes, examples, random buttons)
  • parseVariables support for textarea-based variable lists
  • Remember inputs via localStorage
  • Integration with `text-to-image-plugin` and `ai-text-plugin`
  • Share link generation with gzip compression and upload fallback
  • Modal + getUserInput helpers for interactive flows

Usage: add generateHTML = {import:gw-framework-plugin-v3} to your generator and provide a `settings` list like in t2i frameworks. The plugin will mount controls, inputs, and outputs automatically.
