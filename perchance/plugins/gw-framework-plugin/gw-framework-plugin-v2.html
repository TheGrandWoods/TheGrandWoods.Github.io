GW Framework Plugin v2

This plugin is a unified, full-featured text-to-image framework built from the ideas in t2i-framework-plugin-v2 and t2i-framework-plugin-moss.

Features:

  • Unified runtime API: window.___gw_v2 (event bus, registry, helpers)
  • Share-link generation with optional gzip compression and upload
  • Input persistence, parseable variables, and prompt templating
  • AI prompt suggestions via ai-text-plugin integration
  • Gallery integration and optional private-save flow
  • Compatibility helpers for existing t2i-based generators
  • Modal and getUserInput helpers for interactive UIs

Usage: in your generator header, set:

generateHTML = {import:gw-framework-plugin-v2}

Then pass a settings list with userInputs, imageOptions, and frameworkConfig as you would with other t2i frameworks. Access the runtime API at window.___gw_v2.
